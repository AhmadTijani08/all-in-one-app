# 🎯 All-In-One App - Ultimate Productivity Suite

A beautiful, fully-responsive, all-in-one web application combining **5 powerful features** in one stunning interface!

## ✨ Features

### 🌤️ **1. Weather App**
- Real-time weather data from OpenWeatherMap API
- Current temperature, humidity, wind speed, pressure, visibility
- 5-day weather forecast
- Auto-geolocation + manual city search
- Beautiful weather icons and cards

### ✅ **2. Smart Todo List**
- Add tasks with deadlines and priority levels
- **Smart filtering**: All, Active, Completed, High Priority
- Statistics dashboard
- **Auto-saves to browser** using localStorage
- Beautiful priority badges (High/Medium/Low)

### 🧠 **3. Quiz App with Timer**
- 5 categories: General, Science, History, Geography, Technology
- 10 questions per quiz
- **60-second countdown timer** with visual warning
- Live progress bar
- Score system with detailed results
- Performance feedback

### 💰 **4. Expense Tracker**
- Income/Expense tracking with 8 categories
- Real-time balance calculation
- Beautiful doughnut chart for spending breakdown
- Filter by month
- **Auto-saves to localStorage**
- Detailed transaction history

### 📝 **5. Notes App (Google Keep style)**
- Create, edit, delete notes
- 5 beautiful colors to choose from
- Real-time search functionality
- **Auto-saves automatically**
- Statistics dashboard
- Responsive grid layout

## 🚀 How to Use

1. Simply open `index.html` in your web browser
2. No installation, no dependencies installation needed
3. All data is saved automatically to your browser
4. Toggle dark/light mode in the header
5. Switch between apps using the navigation tabs
